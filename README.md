## ꒰🌷꒱  Sobre o projeto
Este projeto consiste em uma tela de login inspirada na interface moderna de plataformas digitais.
Foi desenvolvido com foco em design visual, organização de layout e aprimoramento das minhas habilidades. Ao longo do projeto, realizei melhorias na estética, buscando uma interface mais agradável e funcional.

## ꒰🌷꒱ Tecnologias utilizadas
- HTML
- CSS
- JSS
- Flexbox
